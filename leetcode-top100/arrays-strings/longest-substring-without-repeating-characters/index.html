    最长无重复子串¶

    题目描述¶

    给定一个字符串 s，请你找出其中不含有重复字符的 最长子串 的长度。

    示例 1：¶

    输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

    示例 2：¶

    输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

    示例 3：¶

    输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。

    提示：¶

    • 0 <= s.length <= 5 * 10⁴
    • s 由英文字母、数字、符号和空格组成

    解题思路¶

    这个问题可以通过两种方法解决： 1. 暴力解法：检查所有可能的子串 2. 滑动窗口：使用双指针和哈希集合优化解法

    最简单的方法（暴力解法）¶

    遍历所有可能的子串，检查它们是否包含重复字符，并记录最长的无重复子串长度。

    def length_of_longest_substring_brute(s):
    if not s:
        return 0

    max_length = 0
    # 遍历所有可能的起始位置
    for i in range(len(s)):
        # 遍历所有可能的结束位置
        for j in range(i, len(s)):
            # 检查子串 s[i:j+1] 是否有重复字符
            substring = s[i:j+1]
            if len(set(substring)) == len(substring):
                # 如果没有重复字符，更新最大长度
                max_length = max(max_length, j - i + 1)
            else:
                # 如果有重复字符，结束当前起始位置的检查
                break

    return max_length

    最优方案（滑动窗口）¶

    使用滑动窗口和哈希集合，可以在一次遍历中找到最长无重复子串。

    def length_of_longest_substring_optimal(s):
    if not s:
        return 0

    # 创建哈希集合，用于存储当前窗口中的字符
    char_set = set()
    max_length = 0
    left = 0

    # 右指针遍历字符串
    for right in range(len(s)):
        # 如果当前字符已经在窗口中，移动左指针直到重复字符被移除
        while s[right] in char_set:
            char_set.remove(s[left])
            left += 1

        # 将当前字符添加到窗口中
        char_set.add(s[right])

        # 更新最大长度
        max_length = max(max_length, right - left + 1)

    return max_length

## 自测用例

```python
# 测试用例1
s1 = "abcabcbb"
print(f"测试用例1: 输入 = {s1}")
print(f"输出: {length_of_longest_substring_optimal(s1)}")  # 预期输出: 3

# 测试用例2
s2 = "bbbbb"
print(f"测试用例2: 输入 = {s2}")
print(f"输出: {length_of_longest_substring_optimal(s2)}")  # 预期输出: 1

# 测试用例3
s3 = "pwwkew"
print(f"测试用例3: 输入 = {s3}")
print(f"输出: {length_of_longest_substring_optimal(s3)}")  # 预期输出: 3

# 测试用例4
s4 = ""
print(f"测试用例4: 输入 = {s4}")
print(f"输出: {length_of_longest_substring_optimal(s4)}")  # 预期输出: 0

# 测试用例5
s5 = "aab"
print(f"测试用例5: 输入 = {s5}")
print(f"输出: {length_of_longest_substring_optimal(s5)}")  # 预期输出: 2
    ```

    复杂度分析¶

    • 暴力解法：
    • 时间复杂度：O(n³)，其中 n 是字符串长度

    • 空间复杂度：O(min(n, m))，其中 m 是字符集大小

    • 滑动窗口解法：

    • 时间复杂度：O(n)，每个字符最多被访问两次
    • 空间复杂度：O(min(n, m))，用于存储字符集

    滑动窗口解法通过一次遍历就可以找到最长无重复子串，避免了不必要的重复计算，大大提高了效率。
